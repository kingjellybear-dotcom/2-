2.(1) 피동 표현과 인용 표현
🔴 피동 표현이란?
주어가 다른 대상에 의해 동작을 당하는 것을 나타낸 표현.
능동문: "경찰이 도둑을 잡았다" → 피동문: "도둑이 경찰에게 잡혔다"
🔴 피동 표현 만드는 법 2가지
방법①파생적 피동 (피동사) — 동사 어근에 -이-, -히-, -리-, -기-를 붙임
  • 보다 → 보이다 / 먹다 → 먹히다 / 팔다 → 팔리다 / 안다 → 안기다

방법②통사적 피동 ('-아/어지다' 붙이기)
  • 읽다 → 읽어지다 / 만들다 → 만들어지다
  ⚠ 피동사가 있는 단어에 '-어지다'를 또 붙이면 이중 피동 → 비문법적! (예: 잡혀지다 ✗)
🔴 피동문의 특징
• 능동문의 목적어 → 피동문의 주어
• 능동문의 주어 → 피동문에서 '-에게', '-에 의해' 등으로 표시 (부사어)
• 피동문은 행위자를 숨기거나 행위의 결과를 강조할 때 사용
🔴 인용 표현 — 직접 인용 vs 간접 인용
구분형식예시
직접 인용큰따옴표(" ") + 라고"빨리 와."라고 했다
간접 인용따옴표 없음 + 고빨리 오라고 했다

간접 인용 시 인칭·시제·종결어미 변환 필수!
예: "나는 행복해" → 자기가 행복하다고 했다 (1인칭→3인칭, 해 체→다 체)

3단원. 부등식과 연립방정식 (p.60~p.142 이전 범위)
📘 부등식 기초 개념
부등식: 두 수 또는 식의 대소 관계를 부등호로 나타낸 식

표현부등호의미
a > b>a는 b보다 크다
a < b<a는 b보다 작다
a ≥ b≥a는 b보다 크거나 같다 (이상)
a ≤ b≤a는 b보다 작거나 같다 (이하)
📘 부등식의 성질 — 시험에 꼭 나옴!
① a < b 이면 a+c < b+c, a-c < b-c (양변에 같은 수 더하기·빼기 → 부등호 방향 유지)
② a < b, c > 0 이면 ac < bc, a/c < b/c (양수 곱/나누기 → 방향 유지)
③ a < b, c < 0 이면 ac > bc, a/c > b/c
⚠ 핵심 주의! 음수를 곱하거나 나눌 때는 반드시 부등호 방향이 뒤집힌다!
📘 일차부등식 풀이법
정의 ax + b < 0 형태 (단, a ≠ 0)

풀이 순서:
① 괄호 있으면 분배법칙으로 먼저 풀기
② x 항은 좌변, 상수는 우변으로 이항
③ 동류항 정리
④ x의 계수로 양변을 나누기 (음수이면 부등호 방향 반전!)

예) 3x - 2 > 7
→ 3x > 9
→ x > 3 ✓

예) -2x + 4 ≤ 10
→ -2x ≤ 6
→ x ≥ -3 ← 음수로 나눴으므로 방향 반전!
📘 일차부등식 활용 문제 유형
유형① 물건 구입: "x개를 사면 a원 이하" → 가격×x ≤ 예산
유형② 속력·거리·시간: 거리=속력×시간 관계 이용
유형③ 평균: (합계) ÷ 횟수 ≥ 목표 점수
유형④ 농도: 소금 양 / 전체 양 × 100

체크! 답이 나왔으면 단위 확인 + 문제 조건에 맞는지 검토 필수
📘 연립방정식 — 미지수 2개, 방정식 2개
연립방정식의 해: 두 방정식을 동시에 만족하는 x, y의 값

가감법 두 방정식을 더하거나 빼서 미지수 하나를 없앰
x + y = 5
x - y = 1
─────── 더하면
2x = 6 → x = 3 → y = 2

대입법 한 식에서 x(또는 y)를 구해 다른 식에 대입
y = 2x - 1 … ①
3x + y = 9 … ②
②에 ① 대입: 3x + (2x-1) = 9
5x = 10 → x = 2, y = 3
📘 연립방정식 해의 특수한 경우
상황그래프해
두 직선 교점 1개한 점에서 만남해가 1개
두 직선 평행만나지 않음해가 없음 (불능)
두 직선 일치완전히 겹침해가 무수히 많음 (부정)
4단원. 일차함수와 그래프 (p.60~p.142)
📘 함수와 일차함수의 정의
함수: x의 값이 정해지면 y의 값이 하나씩 정해지는 관계 → y = f(x)

일차함수: y = ax + b (a ≠ 0 — a가 0이면 상수함수, 일차함수 아님!)
• a = 기울기 (slope), b = y절편
📘 기울기 완벽 이해
기울기(a) = (y의 증가량) ÷ (x의 증가량)
= (y₂ - y₁) ÷ (x₂ - x₁)
두 점 (x₁,y₁), (x₂,y₂)를 지날 때 사용

a 값그래프 모양
a > 0오른쪽 위 방향 (증가)
a < 0오른쪽 아래 방향 (감소)
|a| 클수록기울기 가파름 (y축에 가까움)
|a| 작을수록기울기 완만 (x축에 가까움)
📘 일차함수 그래프 그리기
방법① y절편(0, b)과 기울기 이용
→ y절편 점 찍기 → 기울기(a/1)만큼 이동해 점 하나 더 찍기 → 직선으로 연결

방법② x=0, y=0 대입해 x절편·y절편 구하기
→ 두 점 (-b/a, 0)과 (0, b) 연결
📘 일차함수의 식 구하기
경우① 기울기 a와 y절편 b를 알 때 → y = ax + b 바로 쓰기

경우② 기울기 a와 한 점 (x₁, y₁)을 알 때
y - y₁ = a(x - x₁) → y = ax + b 정리

경우③ 두 점 (x₁,y₁), (x₂,y₂)를 알 때
→ 먼저 기울기 a 계산 → 경우② 방법 적용
📘 일차함수와 일차방정식의 관계
• ax + by + c = 0 형태의 일차방정식을 y에 대해 풀면 일차함수 식이 됨
• x = p: x축에 수직인 직선 (기울기 없음, y축과 평행)
• y = q: x축에 평행인 수평 직선 (기울기 = 0)

연립방정식의 해 = 두 직선의 교점의 좌표

Lesson 1 — New Year, New Me
🔵 핵심 문법: will vs be going to
willbe going to
의미즉흥적 결정, 미래 예측이미 계획한 일, 근거 있는 예측
형태will + 동사원형am/are/is going to + 동사원형
예문I'll answer it. (전화 울릴 때)I'm going to study abroad. (계획)

Lesson 3 — New Ideas Bring Big Changes
🔵 핵심 문법 ①: 수동태
형태: be동사 + 과거분사(p.p.) [+ by 행위자]
의미: 주어가 ~당하다 / ~되다

Lesson 4 — Open the Doors to the World
🔵 핵심 문법 ①: to부정사 vs 동명사
동사목적어 형태예
want, hope, plan, decide, expectto부정사만I want to travel abroad.
enjoy, finish, mind, avoid, keep동명사만I enjoy meeting people.
like, love, hate, start, begin둘 다 가능I like to swim / swimming.
헷갈리는 동사! remember / forget / stop + to부정사 vs 동명사는 의미가 달라짐
• I remember to call him. (앞으로 전화할 것을 기억함)
• I remember calling him. (전화했던 것을 기억함)
