2.(1) 피동 표현과 인용 표현
🔴 피동 표현이란?
주어가 다른 대상에 의해 동작을 당하는 것을 나타낸 표현.
능동문: "경찰이 도둑을 잡았다" → 피동문: "도둑이 경찰에게 잡혔다"
🔴 피동 표현 만드는 법 2가지
방법①파생적 피동 (피동사) — 동사 어근에 -이-, -히-, -리-, -기-를 붙임
  • 보다 → 보이다 / 먹다 → 먹히다 / 팔다 → 팔리다 / 안다 → 안기다

방법②통사적 피동 ('-아/어지다' 붙이기)
  • 읽다 → 읽어지다 / 만들다 → 만들어지다
  ⚠ 피동사가 있는 단어에 '-어지다'를 또 붙이면 이중 피동 → 비문법적! (예: 잡혀지다 ✗)
🔴 피동문의 특징
• 능동문의 목적어 → 피동문의 주어
• 능동문의 주어 → 피동문에서 '-에게', '-에 의해' 등으로 표시 (부사어)
• 피동문은 행위자를 숨기거나 행위의 결과를 강조할 때 사용
🔴 인용 표현 — 직접 인용 vs 간접 인용
구분형식예시
직접 인용큰따옴표(" ") + 라고"빨리 와."라고 했다
간접 인용따옴표 없음 + 고빨리 오라고 했다

간접 인용 시 인칭·시제·종결어미 변환 필수!
예: "나는 행복해" → 자기가 행복하다고 했다 (1인칭→3인칭, 해 체→다 체)

3단원. 부등식과 연립방정식 (p.60~p.142 이전 범위)
📘 부등식 기초 개념
부등식: 두 수 또는 식의 대소 관계를 부등호로 나타낸 식

표현부등호의미
a > b>a는 b보다 크다
a < b<a는 b보다 작다
a ≥ b≥a는 b보다 크거나 같다 (이상)
a ≤ b≤a는 b보다 작거나 같다 (이하)
📘 부등식의 성질 — 시험에 꼭 나옴!
① a < b 이면 a+c < b+c, a-c < b-c (양변에 같은 수 더하기·빼기 → 부등호 방향 유지)
② a < b, c > 0 이면 ac < bc, a/c < b/c (양수 곱/나누기 → 방향 유지)
③ a < b, c < 0 이면 ac > bc, a/c > b/c
⚠ 핵심 주의! 음수를 곱하거나 나눌 때는 반드시 부등호 방향이 뒤집힌다!
📘 일차부등식 풀이법
정의 ax + b < 0 형태 (단, a ≠ 0)

풀이 순서:
① 괄호 있으면 분배법칙으로 먼저 풀기
② x 항은 좌변, 상수는 우변으로 이항
③ 동류항 정리
④ x의 계수로 양변을 나누기 (음수이면 부등호 방향 반전!)

예) 3x - 2 > 7
→ 3x > 9
→ x > 3 ✓

예) -2x + 4 ≤ 10
→ -2x ≤ 6
→ x ≥ -3 ← 음수로 나눴으므로 방향 반전!
📘 일차부등식 활용 문제 유형
유형① 물건 구입: "x개를 사면 a원 이하" → 가격×x ≤ 예산
유형② 속력·거리·시간: 거리=속력×시간 관계 이용
유형③ 평균: (합계) ÷ 횟수 ≥ 목표 점수
유형④ 농도: 소금 양 / 전체 양 × 100

체크! 답이 나왔으면 단위 확인 + 문제 조건에 맞는지 검토 필수
📘 연립방정식 — 미지수 2개, 방정식 2개
연립방정식의 해: 두 방정식을 동시에 만족하는 x, y의 값

가감법 두 방정식을 더하거나 빼서 미지수 하나를 없앰
x + y = 5
x - y = 1
─────── 더하면
2x = 6 → x = 3 → y = 2

대입법 한 식에서 x(또는 y)를 구해 다른 식에 대입
y = 2x - 1 … ①
3x + y = 9 … ②
②에 ① 대입: 3x + (2x-1) = 9
5x = 10 → x = 2, y = 3
📘 연립방정식 해의 특수한 경우
상황그래프해
두 직선 교점 1개한 점에서 만남해가 1개
두 직선 평행만나지 않음해가 없음 (불능)
두 직선 일치완전히 겹침해가 무수히 많음 (부정)
4단원. 일차함수와 그래프 (p.60~p.142)
📘 함수와 일차함수의 정의
함수: x의 값이 정해지면 y의 값이 하나씩 정해지는 관계 → y = f(x)

일차함수: y = ax + b (a ≠ 0 — a가 0이면 상수함수, 일차함수 아님!)
• a = 기울기 (slope), b = y절편
📘 기울기 완벽 이해
기울기(a) = (y의 증가량) ÷ (x의 증가량)
= (y₂ - y₁) ÷ (x₂ - x₁)
두 점 (x₁,y₁), (x₂,y₂)를 지날 때 사용

a 값그래프 모양
a > 0오른쪽 위 방향 (증가)
a < 0오른쪽 아래 방향 (감소)
|a| 클수록기울기 가파름 (y축에 가까움)
|a| 작을수록기울기 완만 (x축에 가까움)
📘 일차함수 그래프 그리기
방법① y절편(0, b)과 기울기 이용
→ y절편 점 찍기 → 기울기(a/1)만큼 이동해 점 하나 더 찍기 → 직선으로 연결

방법② x=0, y=0 대입해 x절편·y절편 구하기
→ 두 점 (-b/a, 0)과 (0, b) 연결
📘 일차함수의 식 구하기
경우① 기울기 a와 y절편 b를 알 때 → y = ax + b 바로 쓰기

경우② 기울기 a와 한 점 (x₁, y₁)을 알 때
y - y₁ = a(x - x₁) → y = ax + b 정리

경우③ 두 점 (x₁,y₁), (x₂,y₂)를 알 때
→ 먼저 기울기 a 계산 → 경우② 방법 적용
📘 일차함수와 일차방정식의 관계
• ax + by + c = 0 형태의 일차방정식을 y에 대해 풀면 일차함수 식이 됨
• x = p: x축에 수직인 직선 (기울기 없음, y축과 평행)
• y = q: x축에 평행인 수평 직선 (기울기 = 0)

🟢 파동의 기본 개념

🟢 빛의 굴절
빛이 한 매질에서 다른 매질로 들어갈 때, 두 매질에서 빛의 속력이 다르기 때문에 경계면에서 진행 방향이 꺾이는 현상
(속력이 같으면 굴절 없이 그대로 직진 — 굴절의 원인은 100% '속력 차이')

상황속력 변화굴절 방향
공기 → 물(또는 유리) (소한 매질→밀한 매질)느려짐법선 쪽으로 꺾임 (굴절각 < 입사각)
물(또는 유리) → 공기 (밀한 매질→소한 매질)빨라짐법선에서 먼 쪽으로 꺾임 (굴절각 > 입사각)
수직으로 입사 (입사각 0°)속력만 변함꺾이지 않고 그대로 직진
⚠ 빛이 빠른 매질로 갈 때 vs 느린 매질로 갈 때
• 느린 매질로 들어갈 때 → 법선에 가까워짐(꺾이는 각이 작아짐)
• 빠른 매질로 들어갈 때 → 법선에서 멀어짐(꺾이는 각이 커짐)
"물 속에 있는 다리가 짧아 보인다", "물에 꽂힌 빨대가 꺾여 보인다", "물고기가 실제 위치보다 떠 보인다" 같은 현상 모두 이 원리!

🟢 굴절 현상 — 자주 나오는 그림 문제 패턴
① 물속에 잠긴 막대가 꺾여 보이는 이유: 막대 아랫부분(물속)에서 나온 빛이 물 표면에서 굴절하여 눈에 들어옴 → 우리 눈은 빛이 꺾이지 않고 직진한 것처럼 인식하여, 실제 위치보다 위쪽(얕은 곳)에 있는 것처럼 보임

🟢 소리의 발생과 전파
소리의 발생: 물체가 진동하면서 주변 매질(공기 등)을 진동시켜 그 진동이 퍼져 나가는 것
소리의 전파: 반드시 매질이 있어야 전파됨 → 진공(매질이 없는 공간)에서는 소리가 전달되지 않음(달에서는 소리가 안 들림)
매질의 종류에 따라 소리의 속력이 다름: 일반적으로 기체 < 액체 < 고체 순으로 빠름 (분자 사이 거리가 가까울수록 진동을 빨리 전달)

구성 입자위치전하비고
양성자원자핵 속+(양전하)원자핵의 무게 대부분 차지

🟢 이온의 형성 원리
원자는 원래 전기적으로 중성이지만, 전자를 잃거나 얻으면 양성자 수와 전자 수가 달라져 전하를 띠게 됨 → 이를 이온이라 함

종류형성 과정전하이온식 표기법
양이온원자가 전자를 잃음(+)원소기호 오른쪽 위에 잃은 전자 수만큼 + 표시
음이온원자가 전자를 얻음(−)원소기호 오른쪽 위에 얻은 전자 수만큼 − 표시
예) 나트륨 원자(Na)가 전자 1개를 잃음 → Na⁺ (나트륨 이온)
예) 염소 원자(Cl)가 전자 1개를 얻음 → Cl⁻ (염화 이온)
예) 마그네슘 원자(Mg)가 전자 2개를 잃음 → Mg²⁺
※ 전하가 1일 때는 숫자 1을 쓰지 않고 +, −만 표기! (Na¹⁺ ✗ → Na⁺ ✓)

🟢 이온 결합과 이온 화합물의 화학식
이온 결합: 양이온과 음이온이 전기적 인력으로 서로 끌어당겨 결합하는 것 → 이온 화합물 생성

화학식 만드는 법 양이온과 음이온의 전체 전하량의 합이 0이 되도록 개수를 맞춤
예) Na⁺ + Cl⁻ → NaCl (전하 +1과 −1이 만나 0)
예) Ca²⁺ + Cl⁻ → CaCl₂ (Ca²⁺ 1개당 Cl⁻ 2개 필요: +2와 −1×2=−2)
예) Na⁺ + OH⁻ → NaOH
예) Mg²⁺ + O²⁻ → MgO (1:1로 전하가 이미 맞음)
⚠ 시험 빈출 유형 "다음 양이온과 음이온으로 만들 수 있는 화학식을 쓰시오" — 전하의 절댓값을 서로 바꿔서 아래 첨자로 쓰면 빠르게 풀 수 있음 (교차법). 단, 이미 약분 가능한 비율이면 간단히 줄여서 쓸 것 (예: Mg²⁺와 O²⁻ → Mg₂O₂가 아니라 MgO)

스파르타 군국주의 체제, 강한 군사 훈련 중심 사회(소수의 시민이 다수의 노예를 통제하기 위함)
페르시아 전쟁 그리스 연합군이 마라톤 전투 등에서 페르시아를 격퇴
펠로폰네소스 전쟁 아테네 vs 스파르타 → 그리스 세계 전체 쇠퇴
헬레니즘 문화 알렉산드로스 대왕의 동방 원정 → 그리스 문화 + 오리엔트(동방) 문화가 융합되어 형성

• 공화정: 원로원 중심, 귀족(파트리키) vs 평민(플레브스) 갈등 → 리키니우스법·호르텐시우스법으로 평민 권리 점차 확대
• 카이사르 → 옥타비아누스 시대를 거쳐 제정 시작(옥타비아누스가 초대 황제 '아우구스투스'가 됨)
• 로마의 업적: 로마법(이후 유럽 법체계의 기초), 도로망("모든 길은 로마로 통한다"), 콜로세움, 판테온

Ⅱ. 동서 문화의 형성 — 동아시아 세계
🟠 춘추 전국 시대 — 혼란 속의 사상 발달
주나라의 봉건제가 무너지면서 여러 나라가 패권을 다툰 분열·전쟁 시기
철기 보급 농업 생산력 증가, 새로운 무기 등장 → 전쟁이 더 격렬해짐
제자백가 혼란을 극복하기 위한 다양한 사상이 등장 — 유가(공자, 인과 예 강조), 도가(노자, 자연스러움 강조), 법가(엄격한 법과 제도로 통치 강조), 묵가 등
💡 이후 진나라가 법가 사상을 바탕으로 통일에 성공함 — 사상과 후속 통일 왕조를 연결해서 기억!
🟠 진나라 — 중국 최초의 통일
진시황제가 춘추 전국의 혼란을 끝내고 중국 최초로 통일(기원전 221년)
통치 정책 법가 사상 기반의 강력한 중앙 집권 체제, 문자·도량형·화폐 통일, 군현제 실시(전국에 관리 직접 파견)
대표 건축 만리장성 축조(북방 민족 침입 방어), 병마용 — 황제 무덤을 지키는 거대한 병사 모형군
멸망 가혹한 통치와 대규모 토목 공사로 백성의 불만 폭증 → 진시황제 사후 빠르게 멸망(통일 후 약 15년)
🟠 한나라 — 동아시아 문화의 기틀
진나라 멸망 후 유방이 건국. 진의 급격한 정책을 완화하며 안정적으로 발전
한 무제 시기 전성기 군현제 강화, 유교를 통치 이념으로 채택(이후 동아시아 전체에 큰 영향), 비단길을 통한 동서 교역 활발
외부 압박 북방 유목 민족 흉노와 계속 대립
🟠 후한 — 한나라의 부흥과 쇠퇴
왕망의 신나라로 잠시 한이 멸망했다가, 광무제가 다시 세운 한 왕조(=후한)
후반기에는 외척과 환관(궁중 내시)의 권력 다툼, 농민 봉기(황건적의 난 등)로 점차 혼란 → 결국 멸망 → 이후 위·진·남북조의 분열기로 이어짐
🟠 위진 남북조 시대 — 분열과 민족 융합
후한 멸망 후 중국이 위·촉·오(삼국)로 분열되었다가, 이후 화북(북방)과 강남(남방)으로 나뉘어 여러 왕조가 교체된 장기 분열기
특징 북방 유목 민족이 화북 지역에 여러 나라를 세움(북조), 한족은 강남으로 이동해 왕조를 이어감(남조) → 한족과 북방 민족의 문화가 섞이는 민족 융합이 진행됨
문화 이 시기 인도에서 전해진 불교가 크게 융성(원강·룽먼 석굴 등 불교 예술 발달)
🟠 수나라 — 짧지만 중요한 재통일
위진 남북조의 오랜 분열을 끝내고 중국을 다시 통일(6세기 말)
업적 대운하 건설 — 강남(경제 중심)과 화북(정치 중심)을 연결해 물자 운송을 원활하게 함
멸망 원인 대운하 등 대규모 토목 공사 + 고구려 원정 실패(살수 대첩 등)로 국력 소모 → 백성 반란으로 단명(진나라와 유사한 패턴!)
🟠 당나라 — 동아시아 문화권의 완성
수나라를 이어 건국, 약 300년간 지속되며 동아시아 최대의 국제적 제국으로 발전
제도 율령(법전) 체제 정비 → 주변국(신라, 일본 등)이 이를 받아들여 각자의 제도를 정비함
국제 도시 장안 비단길의 중심지로 다양한 외국 상인·종교(이슬람교, 조로아스터교 등)가 공존하는 국제적인 수도였음
동아시아 문화권 한자, 율령, 유교, 불교를 공유하는 문화권이 당을 중심으로 완성됨 → 한국·일본·베트남 등에 큰 영향
쇠퇴 안·사의 난 이후 지방 세력(절도사) 강화 → 점차 쇠퇴
🟠 한국의 고대 국가 형성
동아시아 문화권의 일원으로서 한반도에서도 고대 국가가 성립·발전
고구려·백제·신라 각각 왕권 강화 및 율령 정비, 불교 수용을 거치며 고대 국가 체제를 완성(삼국 시대)
통일과 발전 신라가 당과 연합해 삼국을 통일 → 이후 발해가 고구려를 계승해 만주·한반도 북부에서 성장 → 신라와 발해가 공존(남북국)
동아시아 문화권과의 관계 당의 율령·유교·불교·한자를 받아들여 독자적으로 발전시킴 — 중국 문화의 단순 수용이 아니라 자기 방식대로 재구성한 점이 핵심 포인트
🟠 일본의 고대 국가
동아시아 문화권의 영향을 받아 일본에서도 고대 국가가 형성됨
다이카 개신 당의 율령 제도를 본떠 중앙 집권 체제를 정비하려 한 개혁
불교 수용 한반도(백제 등)를 통해 불교·한자 등 대륙 문화가 전해짐 — 아스카 문화 발달의 토대
나라 시대 당의 장안성을 본떠 헤이조쿄(나라)를 수도로 건설, 본격적인 율령 국가 체제 확립
Ⅱ. 동서 문화의 형성 — 인도와 남아시아
🟠 사산 왕조 페르시아
아키메네스 왕조 이후 다시 페르시아 지역을 통일한 왕조
위치와 역할 동서 교역로(비단길)의 중심에 위치해 동서 문화 교류의 통로 역할 → 비잔티움 제국과는 오랜 경쟁 관계
종교·문화 조로아스터교를 국교로 삼아 발전시킴, 페르시아 고유 문화를 유지하며 동서 문화 융합에 기여
🟠 마우리아 왕조 — 인도 최초의 통일 제국
인도 지역을 최초로 통일한 왕조
아소카왕 시기 전성기 정복 전쟁의 참혹함을 반성하고 불교에 귀의 → 불교를 통한 통치 이념 확립, 각지에 불교 사원·돌기둥(석주) 건립, 불교를 인도 밖(스리랑카 등)으로 전파
🟠 쿠샨 왕조 — 간다라 미술
마우리아 왕조 이후 인도 북부에서 성장, 동서 교역(비단길)으로 번영
불교의 변화 부처를 신앙의 대상으로 삼는 대승 불교가 발전 → 동아시아(중국·한국·일본)로 전파되는 불교는 이 계통
간다라 미술 그리스(헬레니즘) 조각 양식 + 인도 불교가 결합 → 사람 모습을 한 불상이 처음으로 만들어짐 (이전에는 부처의 모습을 직접 형상화하지 않았음)
🟠 굽타 왕조 — 힌두 문화의 완성
쿠샨 왕조 이후 인도를 다시 통일, 인도 고전 문화의 황금기를 이룸
힌두교의 발전 브라만교를 바탕으로 민간 신앙과 불교 요소가 결합되어 힌두교가 체계화되고 인도 사회 전반에 정착
문화 성과 산스크리트 문학(『마하바라타』 등), 숫자 '0'의 개념과 10진법 체계 정립 → 이후 이슬람을 거쳐 유럽에 전파(아라비아 숫자의 기원)
🟠 힌두교 — 인도 사회와 결합한 종교
다신교적 성격(브라흐마·비슈누·시바 등 여러 신을 섬김), 카르마(업)와 환생 사상이 핵심
카스트 제도와 깊게 결합되어 인도 사회 신분 질서를 정당화하는 역할도 함 → 카스트(브라만-크샤트리아-바이샤-수드라)와 힌두교의 관계를 묻는 문제 자주 출제됨
Ⅲ. 지역 세계의 교류와 변화 — 유럽과 이슬람
🟠 비잔티움 제국
로마 제국이 동서로 분열된 후, 동로마(콘스탄티노폴리스 수도) 제국이 약 1000년간 존속하며 그리스·로마 문화를 계승
유스티니아누스 황제 옛 로마 영토 일부 회복, 『유스티니아누스 법전』 편찬(로마법 집대성)
문화 그리스 정교(크리스트교의 한 갈래) 발달, 화려한 모자이크·돔 건축(성 소피아 대성당)
역할 서유럽과 이슬람 세계 사이에서 그리스·로마 고전 문화를 보존·중계하는 역할 → 이후 서유럽 문화 부흥(르네상스)에 영향
🟠 프랑크 왕국 — 서유럽 중세의 시작
서로마 제국 멸망 후 서유럽에서 성장한 게르만 왕국 중 가장 크게 성공
크리스트교 수용 로마 가톨릭으로 개종 → 로마 교회와 협력 관계 구축(다른 게르만족과 다른 길)
카롤루스 대제 서유럽 대부분 통합, 로마 교황으로부터 서로마 황제 대관 → 이후 왕국이 분열되며 오늘날 프랑스·독일·이탈리아의 기원이 됨
🟠 중세 유럽의 봉건제와 교회
• 봉건제: 왕 → 영주 → 기사 → 농노로 이어지는 계층 구조, 주군-봉신 사이의 쌍방적 계약 관계(충성 서약 + 보호 의무)
• 로마 가톨릭 교회의 강력한 영향력 — 한때 교황권이 황제권보다 우위에 섬
• 카노사의 굴욕(1077): 황제 하인리히 4세가 교황에게 무릎 꿇고 사죄 → 교황권이 최고조에 이른 사건
• 십자군 전쟁: 성지 예루살렘 탈환을 명분으로 시작, 약 200년간 7차례 전개 → 결국 실패 → 교황권 약화, 왕권 강화, 동서 교류 활발(이슬람·비잔티움 문화 유입)
🟠 이슬람 세계의 형성과 발전
• 무함마드(마호메트): 7세기 아라비아 반도에서 이슬람교 창시(유일신 알라, 경전 코란)
• 짧은 기간에 아라비아 반도 → 중동·북아프리카·중앙아시아까지 급속 팽창
• 이슬람 문화: 아라비아 숫자(인도 숫자 체계 수용·전파), 대수학(알제브라), 연금술, 의학 발전 — 그리스·로마, 인도, 페르시아 문화를 폭넓게 수용해 독자적으로 발전시킨 점이 특징
• 동서 무역의 중계 역할 수행(비단길, 바닷길 모두 장악)
Ⅲ. 지역 세계의 교류와 변화 — 북방 민족과 송·원
🟠 송나라 — 문치주의와 경제 발전
당 멸망 후 5대 10국의 혼란을 수습하며 건국
문치주의 무인보다 문인(과거 시험으로 선발된 관료)을 우대하는 정책 → 군사력은 상대적으로 약화됨
경제·문화 발전 강남 지역 농업 생산력 증가, 상업·도시 발달(지폐 사용 시작), 성리학 등장(주희가 체계화)
대외 관계의 약점 군사적으로 약해 북방 민족(거란, 서하, 여진)에게 막대한 비단·은을 주며 평화를 유지하는 경우가 많았음 — 문치주의의 명암을 함께 기억할 것!
🟠 거란(요) — 송을 압박한 북방 민족
유목 민족인 거란족이 세운 나라(요)
송과 군사적으로 대립하다가 화의를 맺어 송으로부터 매년 막대한 물품(비단·은)을 받기로 함 → 이후 동아시아 북방 민족들이 송을 압박해 경제적 이익을 얻는 패턴의 시초
🟠 서하 — 송·거란 사이의 또 다른 압박
탕구트족이 중국 서북부에 세운 나라
독자적인 문자(서하 문자)를 만들어 사용했으며, 송에 압박을 가해 거란과 비슷하게 물품을 받아내는 등 송을 둘러싼 다중 압박 구도를 형성함(거란·서하·송의 삼각 관계)
🟠 여진(금) — 송을 더 강하게 압박
만주 지역의 여진족이 거란(요)을 멸망시키고 세운 나라(금)
이후 송을 더욱 강하게 압박해 화북 지역을 차지 → 송은 강남으로 밀려나 남송으로 이어짐(송 왕조의 영토가 절반으로 축소되는 결과)
⚠ 북방 민족 순서 기억법 거란(요) → 서하 → 여진(금) → (이후) 몽골 순으로 등장하며, 뒤로 갈수록 송에 대한 압박이 점점 강해진다는 흐름을 기억!
🟠 몽골 제국과 유라시아 교류
• 칭기즈 칸: 13세기 몽골 초원의 부족들을 통일 → 거대한 정복 전쟁 시작
• 짧은 시간에 유라시아 대부분을 지배하는 사상 최대 규모의 제국 건설
• 팍스 몽골리카(몽골에 의한 평화): 광대한 영토를 하나로 묶어 동서 교역과 문화 교류가 전례 없이 활발해진 시기 — 비단길이 안전하게 정비되어 상인·여행가의 왕래가 폭증
• 원나라(쿠빌라이 칸이 건국): 남송을 멸망시키고 중국 전체 지배 / 마르코 폴로가 방문해 『동방견문록』을 남김
• 부정적 영향: 흑사병(페스트)이 활발해진 교류망(실크로드)을 따라 유럽까지 전파되어 인구가 크게 감소함
유라시아 교류의 의미 몽골 제국 시기를 거치며 동서양이 하나의 교역망으로 연결되는 경험을 함 — 이후 세계화의 초기 형태로 평가됨
Ⅳ. 동아시아와 인도지역 질서의 변화
🟠 명·청 교체와 동아시아
• 명나라(주원장 건국): 몽골(원)을 북으로 몰아내고 한족 왕조 부활 / 정화의 대원정(인도양·아프리카까지 진출하는 대규모 해상 탐험) / 만리장성 대대적 보수
• 청나라(여진족이 세운 후금 → 청으로 발전, 이후 만주족): 명 멸망 후 중국 전체를 지배 / 한족에게는 변발·호복(만주족 복식)을 강요하면서도 한족의 제도와 문화를 적극 수용하는 이중 정책
→ 강희제·옹정제·건륭제 시기에 최고 전성기를 맞음(영토 최대 확장, 안정적 통치)
🟠 가마쿠라 막부 — 일본 최초의 무사 정권
미나모토 요리토모가 수립한 일본 최초의 막부(무사 정권)
통치 구조 천황은 명목상 존재만 유지, 실질적인 권력은 쇼군(막부의 우두머리)이 행사하는 이중 권력 구조 시작 — 이후 일본 역사에서 막부 체제가 약 700년간 지속되는 출발점
대외 관계 몽골(원)의 두 차례 침입을 받았으나 태풍(가미카제) 등의 도움으로 격퇴
🟠 무로마치 막부
가마쿠라 막부 멸망 후 아시카가 가문이 수립한 막부
대외 교류 명나라와 조공·무역 관계를 맺어 활발히 교역(왜구 단속의 대가로 무역 허가를 받음)
쇠퇴 점차 쇼군의 권력이 약화되고 지방 영주(다이묘)들의 세력이 강해지면서 중앙 통제력을 잃어감 → 이후 일본 전국 시대로 이어짐
🟠 일본 전국 시대 — 군웅할거의 시대
무로마치 막부의 권위가 무너지면서 각지의 다이묘(지방 영주)들이 독자적으로 세력을 다투던 약 100여 년간의 분열·전쟁 시기
통일 과정 오다 노부나가가 통일의 기반을 마련 → 도요토미 히데요시가 일본을 통일 → 이후 도쿠가와 이에야스가 에도 막부를 세우며 전국 시대를 완전히 종식
💡 도요토미 히데요시의 일본 통일 이후 조선을 침략한 것이 임진왜란(1592)이라는 점도 흐름상 함께 기억해두면 좋음

🔵 핵심 문법 ①: 수동태
형태: be동사 + 과거분사(p.p.) [+ by 행위자]
의미: 주어가 ~당하다 / ~되다

능동: Edison invented the light bulb.
수동: The light bulb was invented by Edison.

시제별 형태:
현재: am/are/is + p.p.
과거: was/were + p.p.
미래: will be + p.p.

선행사관계대명사역할

사물/동물which / that주격·목적격

예: The app that I use every day is very useful.

Lesson 4 — Open the Doors to the World
🔵 핵심 문법 ①: to부정사 vs 동명사
동사목적어 형태예
want, hope, plan, decide, expectto부정사만I want to travel abroad.
enjoy, finish, mind, avoid, keep동명사만I enjoy meeting people.
like, love, hate, start, begin둘 다 가능I like to swim / swimming.
헷갈리는 동사! remember / forget / stop + to부정사 vs 동명사는 의미가 달라짐
• I remember to call him. (앞으로 전화할 것을 기억함)
• I remember calling him. (전화했던 것을 기억함)
🔵 핵심 문법 ②: 접속사·접속부사
접속사/접속부사의미예
although / though비록 ~이지만Although it was cold, we went out.
because / since / as~때문에I stayed home because it rained.
however그러나 (문장 연결)It was expensive. However, I bought it.
therefore따라서He studied hard. Therefore, he passed.
