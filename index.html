2.(1) 피동 표현과 인용 표현
🔴 피동 표현이란?
주어가 다른 대상에 의해 동작을 당하는 것을 나타낸 표현.
능동문: "경찰이 도둑을 잡았다" → 피동문: "도둑이 경찰에게 잡혔다"
🔴 피동 표현 만드는 법 2가지
방법①파생적 피동 (피동사) — 동사 어근에 -이-, -히-, -리-, -기-를 붙임
  • 보다 → 보이다 / 먹다 → 먹히다 / 팔다 → 팔리다 / 안다 → 안기다

방법②통사적 피동 ('-아/어지다' 붙이기)
  • 읽다 → 읽어지다 / 만들다 → 만들어지다
  ⚠ 피동사가 있는 단어에 '-어지다'를 또 붙이면 이중 피동 → 비문법적! (예: 잡혀지다 ✗)
🔴 피동문의 특징
• 능동문의 목적어 → 피동문의 주어
• 능동문의 주어 → 피동문에서 '-에게', '-에 의해' 등으로 표시 (부사어)
• 피동문은 행위자를 숨기거나 행위의 결과를 강조할 때 사용
🔴 인용 표현 — 직접 인용 vs 간접 인용
구분형식예시
직접 인용큰따옴표(" ") + 라고"빨리 와."라고 했다
간접 인용따옴표 없음 + 고빨리 오라고 했다

간접 인용 시 인칭·시제·종결어미 변환 필수!
예: "나는 행복해" → 자기가 행복하다고 했다 (1인칭→3인칭, 해 체→다 체)

3단원. 부등식과 연립방정식 (p.60~p.142 이전 범위)
📘 부등식 기초 개념
부등식: 두 수 또는 식의 대소 관계를 부등호로 나타낸 식

표현부등호의미
a > b>a는 b보다 크다
a < b<a는 b보다 작다
a ≥ b≥a는 b보다 크거나 같다 (이상)
a ≤ b≤a는 b보다 작거나 같다 (이하)
📘 부등식의 성질 — 시험에 꼭 나옴!
① a < b 이면 a+c < b+c, a-c < b-c (양변에 같은 수 더하기·빼기 → 부등호 방향 유지)
② a < b, c > 0 이면 ac < bc, a/c < b/c (양수 곱/나누기 → 방향 유지)
③ a < b, c < 0 이면 ac > bc, a/c > b/c
⚠ 핵심 주의! 음수를 곱하거나 나눌 때는 반드시 부등호 방향이 뒤집힌다!
📘 일차부등식 풀이법
정의 ax + b < 0 형태 (단, a ≠ 0)

풀이 순서:
① 괄호 있으면 분배법칙으로 먼저 풀기
② x 항은 좌변, 상수는 우변으로 이항
③ 동류항 정리
④ x의 계수로 양변을 나누기 (음수이면 부등호 방향 반전!)

예) 3x - 2 > 7
→ 3x > 9
→ x > 3 ✓

예) -2x + 4 ≤ 10
→ -2x ≤ 6
→ x ≥ -3 ← 음수로 나눴으므로 방향 반전!
📘 일차부등식 활용 문제 유형
유형① 물건 구입: "x개를 사면 a원 이하" → 가격×x ≤ 예산
유형② 속력·거리·시간: 거리=속력×시간 관계 이용
유형③ 평균: (합계) ÷ 횟수 ≥ 목표 점수
유형④ 농도: 소금 양 / 전체 양 × 100

체크! 답이 나왔으면 단위 확인 + 문제 조건에 맞는지 검토 필수
📘 연립방정식 — 미지수 2개, 방정식 2개
연립방정식의 해: 두 방정식을 동시에 만족하는 x, y의 값

가감법 두 방정식을 더하거나 빼서 미지수 하나를 없앰
x + y = 5
x - y = 1
─────── 더하면
2x = 6 → x = 3 → y = 2

대입법 한 식에서 x(또는 y)를 구해 다른 식에 대입
y = 2x - 1 … ①
3x + y = 9 … ②
②에 ① 대입: 3x + (2x-1) = 9
5x = 10 → x = 2, y = 3
📘 연립방정식 해의 특수한 경우
상황그래프해
두 직선 교점 1개한 점에서 만남해가 1개
두 직선 평행만나지 않음해가 없음 (불능)
두 직선 일치완전히 겹침해가 무수히 많음 (부정)
4단원. 일차함수와 그래프 (p.60~p.142)
📘 함수와 일차함수의 정의
함수: x의 값이 정해지면 y의 값이 하나씩 정해지는 관계 → y = f(x)

일차함수: y = ax + b (a ≠ 0 — a가 0이면 상수함수, 일차함수 아님!)
• a = 기울기 (slope), b = y절편
📘 기울기 완벽 이해
기울기(a) = (y의 증가량) ÷ (x의 증가량)
= (y₂ - y₁) ÷ (x₂ - x₁)
두 점 (x₁,y₁), (x₂,y₂)를 지날 때 사용

a 값그래프 모양
a > 0오른쪽 위 방향 (증가)
a < 0오른쪽 아래 방향 (감소)
|a| 클수록기울기 가파름 (y축에 가까움)
|a| 작을수록기울기 완만 (x축에 가까움)
📘 일차함수 그래프 그리기
방법① y절편(0, b)과 기울기 이용
→ y절편 점 찍기 → 기울기(a/1)만큼 이동해 점 하나 더 찍기 → 직선으로 연결

방법② x=0, y=0 대입해 x절편·y절편 구하기
→ 두 점 (-b/a, 0)과 (0, b) 연결
📘 일차함수의 식 구하기
경우① 기울기 a와 y절편 b를 알 때 → y = ax + b 바로 쓰기

경우② 기울기 a와 한 점 (x₁, y₁)을 알 때
y - y₁ = a(x - x₁) → y = ax + b 정리

경우③ 두 점 (x₁,y₁), (x₂,y₂)를 알 때
→ 먼저 기울기 a 계산 → 경우② 방법 적용
📘 일차함수와 일차방정식의 관계
• ax + by + c = 0 형태의 일차방정식을 y에 대해 풀면 일차함수 식이 됨
• x = p: x축에 수직인 직선 (기울기 없음, y축과 평행)
• y = q: x축에 평행인 수평 직선 (기울기 = 0)

🟢 파동의 기본 개념
파동: 한 곳에서 생긴 진동(에너지)이 매질을 통해 주위로 퍼져 나가는 현상
→ 매질 자체는 이동하지 않고 제자리에서 진동만 한다! (물 위에 띄운 종이배는 제자리에서 위아래로만 흔들림 — 파도를 따라 이동 ✗)

요소정의단위
마루파동에서 가장 높은 부분-
골파동에서 가장 낮은 부분-
진폭진동의 중심(평형점)에서 마루(또는 골)까지의 거리m, cm
파장(λ)마루에서 다음 마루까지(또는 골~골) 거리 = 파동 1개의 길이m, cm
진동수(f)1초 동안 매질이 진동하는 횟수Hz
주기(T)매질이 한 번 진동하는 데 걸리는 시간s
파동의 속력(v) = 파장(λ) ÷ 주기(T) = 파장(λ) × 진동수(f)
주기(T) = 1 ÷ 진동수(f)  |  진동수(f) = 1 ÷ 주기(T)
⚠ 그래프 문제 주의! '변위-시간 그래프'에서는 가로축 한 칸이 곧 주기(T)와 연결되고, '변위-거리(위치) 그래프'에서는 가로축 한 칸이 파장(λ)과 연결됨. 두 그래프를 헷갈리면 진동수·파장을 바꿔 계산하는 실수가 가장 많이 나옴 — 그래프 종류부터 확인!
💡 진폭이 클수록 파동이 가진 에너지가 크다 (소리라면 더 크게 들림, 빛이라면 더 밝게 보임)
🟢 빛의 반사 법칙 — 정확한 정의부터!
법선: 거울(반사면)에 수직으로 그은 가상의 선 — 입사각·반사각은 항상 거울면이 아니라 법선을 기준으로 측정!
입사각: 입사 광선과 법선 사이의 각 / 반사각: 반사 광선과 법선 사이의 각

반사 법칙: 입사각 = 반사각 (항상 성립, 예외 없음)
⚠ 자주 틀리는 포인트 문제에서 거울면과 광선이 이루는 각을 줬을 때, 이건 입사각이 아님! 90°에서 빼야 진짜 입사각(법선 기준)이 나옴. 예) 거울면과 50° → 입사각은 90°−50°=40°

🟢 빛의 굴절
빛이 한 매질에서 다른 매질로 들어갈 때, 두 매질에서 빛의 속력이 다르기 때문에 경계면에서 진행 방향이 꺾이는 현상
(속력이 같으면 굴절 없이 그대로 직진 — 굴절의 원인은 100% '속력 차이')

상황속력 변화굴절 방향
공기 → 물(또는 유리) (소한 매질→밀한 매질)느려짐법선 쪽으로 꺾임 (굴절각 < 입사각)
물(또는 유리) → 공기 (밀한 매질→소한 매질)빨라짐법선에서 먼 쪽으로 꺾임 (굴절각 > 입사각)
수직으로 입사 (입사각 0°)속력만 변함꺾이지 않고 그대로 직진
⚠ 빛이 빠른 매질로 갈 때 vs 느린 매질로 갈 때
• 느린 매질로 들어갈 때 → 법선에 가까워짐(꺾이는 각이 작아짐)
• 빠른 매질로 들어갈 때 → 법선에서 멀어짐(꺾이는 각이 커짐)
"물 속에 있는 다리가 짧아 보인다", "물에 꽂힌 빨대가 꺾여 보인다", "물고기가 실제 위치보다 떠 보인다" 같은 현상 모두 이 원리!
단색광 분산 프리즘에 백색광(태양광)을 통과시키면 색마다 굴절하는 정도가 달라 무지개色으로 분리됨 (빨강은 적게, 보라는 많이 꺾임)
🟢 굴절 현상 — 자주 나오는 그림 문제 패턴
① 물속에 잠긴 막대가 꺾여 보이는 이유: 막대 아랫부분(물속)에서 나온 빛이 물 표면에서 굴절하여 눈에 들어옴 → 우리 눈은 빛이 꺾이지 않고 직진한 것처럼 인식하여, 실제 위치보다 위쪽(얕은 곳)에 있는 것처럼 보임

② 동전이 실제보다 떠 보이는 이유: 물 속 동전에서 나온 빛이 물 표면에서 공기 쪽으로 굴절(법선에서 멀어짐) → 그 빛의 연장선을 따라가면 실제 동전보다 얕은 곳에 동전이 있는 것처럼 느껴짐

③ 작도 순서: ⓐ 입사 광선 그리기 → ⓑ 경계면에 법선 긋기 → ⓒ 매질 밀도 비교(밀한 쪽으로 갈 때는 법선 쪽으로, 소한 쪽으로 갈 때는 법선에서 먼 쪽으로) → ⓓ 굴절 광선 그리기
🟢 볼록 렌즈 — 물체 위치별 상 (★최다 출제, 오목 거울과 패턴 유사)

🟢 소리의 발생과 전파
소리의 발생: 물체가 진동하면서 주변 매질(공기 등)을 진동시켜 그 진동이 퍼져 나가는 것
소리의 전파: 반드시 매질이 있어야 전파됨 → 진공(매질이 없는 공간)에서는 소리가 전달되지 않음(달에서는 소리가 안 들림)
매질의 종류에 따라 소리의 속력이 다름: 일반적으로 기체 < 액체 < 고체 순으로 빠름 (분자 사이 거리가 가까울수록 진동을 빨리 전달)
🟢 소리의 3요소 (★최다 출제 — 그래프 해석 필수)
요소결정 요인그래프에서 비교
크기(음량)진폭진폭이 클수록 큰 소리 (그래프 세로 폭이 큼)
높낮이(음고)진동수진동수가 클수록(파형이 촘촘할수록) 높은 소리
음색파형의 모양같은 높이·크기여도 파형 생김새가 다르면 다른 악기·다른 사람 목소리로 들림
⚠ 그래프 비교 문제 풀이 순서
① 세로축(진폭) 비교 → 큰 소리/작은 소리 판단
② 가로축 한 파장 안의 진동 횟수(촘촘함) 비교 → 높은 소리/낮은 소리 판단
③ 파형의 모양 비교 → 같은 음인지, 다른 소리(음색)인지 판단
세 가지를 한 그래프에서 동시에 물어보는 문제가 가장 많이 출제됨!
현악기 예 줄을 팽팽하게 조일수록·짧게 잡을수록 진동수↑ → 높은 소리
관악기 예 관의 길이가 짧을수록 진동수↑ → 높은 소리

🟢 원자의 구조 (★최다 출제)
원자: 물질을 이루는 기본 입자. 중심의 원자핵과 그 주위를 도는 전자로 구성

구성 입자위치전하비고
양성자원자핵 속+(양전하)원자핵의 무게 대부분 차지
중성자원자핵 속전하 없음(중성)원자핵의 무게 대부분 차지
전자원자핵 주위(전자껍질)−(음전하)질량이 매우 작아 무시 가능
원자 번호 = 양성자 수
중성 원자에서는 양성자 수 = 전자 수 (전체 전하량이 0이 되어야 하므로)
질량수 = 양성자 수 + 중성자 수
⚠ 가장 많이 나오는 함정 "원자는 양성자 수와 전자 수가 같아서 전기적으로 중성"이라는 문장 → 이온이 되면(전자를 잃거나 얻으면) 이 균형이 깨져서 전하를 띠게 됨! 원자=중성, 이온=전하 띤 입자라는 차이를 명확히 구분.
🟢 분자 — 물질의 성질을 갖는 가장 작은 입자
분자: 원자가 일정한 수로 결합하여 만들어진, 독립적인 입자로 존재하며 물질의 성질을 나타내는 가장 작은 단위
(원자 그 자체는 물질의 성질을 가지지 못하는 경우가 많음 — 예: 수소 원자 1개만으로는 수소 기체의 성질을 보이지 않고, H₂ 분자 상태여야 함)

🟢 이온의 형성 원리
원자는 원래 전기적으로 중성이지만, 전자를 잃거나 얻으면 양성자 수와 전자 수가 달라져 전하를 띠게 됨 → 이를 이온이라 함

종류형성 과정전하이온식 표기법
양이온원자가 전자를 잃음(+)원소기호 오른쪽 위에 잃은 전자 수만큼 + 표시
음이온원자가 전자를 얻음(−)원소기호 오른쪽 위에 얻은 전자 수만큼 − 표시
예) 나트륨 원자(Na)가 전자 1개를 잃음 → Na⁺ (나트륨 이온)
예) 염소 원자(Cl)가 전자 1개를 얻음 → Cl⁻ (염화 이온)
예) 마그네슘 원자(Mg)가 전자 2개를 잃음 → Mg²⁺
※ 전하가 1일 때는 숫자 1을 쓰지 않고 +, −만 표기! (Na¹⁺ ✗ → Na⁺ ✓)

🟢 이온 결합과 이온 화합물의 화학식
이온 결합: 양이온과 음이온이 전기적 인력으로 서로 끌어당겨 결합하는 것 → 이온 화합물 생성

화학식 만드는 법 양이온과 음이온의 전체 전하량의 합이 0이 되도록 개수를 맞춤
예) Na⁺ + Cl⁻ → NaCl (전하 +1과 −1이 만나 0)
예) Ca²⁺ + Cl⁻ → CaCl₂ (Ca²⁺ 1개당 Cl⁻ 2개 필요: +2와 −1×2=−2)
예) Na⁺ + OH⁻ → NaOH
예) Mg²⁺ + O²⁻ → MgO (1:1로 전하가 이미 맞음)
⚠ 시험 빈출 유형 "다음 양이온과 음이온으로 만들 수 있는 화학식을 쓰시오" — 전하의 절댓값을 서로 바꿔서 아래 첨자로 쓰면 빠르게 풀 수 있음 (교차법). 단, 이미 약분 가능한 비율이면 간단히 줄여서 쓸 것 (예: Mg²⁺와 O²⁻ → Mg₂O₂가 아니라 MgO)
🟢 이온의 확인 — 앙금 생성 반응 (★단원 마지막 핵심)

🟢 순물질과 혼합물
구분정의예
순물질 - 홑원소 물질한 종류의 원소로만 이루어짐산소(O₂), 철(Fe), 금(Au), 다이아몬드(C)
순물질 - 화합물2종 이상의 원소가 일정한 비율로 화학적 결합물(H₂O), 소금(NaCl), 이산화탄소(CO₂)
혼합물2종 이상의 순물질이 일정하지 않은 비율로 물리적으로 섞임공기, 소금물, 합금(황동·청동), 우유
⚠ 화합물 vs 혼합물 구분 핵심
• 화합물: 화학 결합으로 새로운 성질의 물질이 됨 (성분 원소의 성질 사라짐, 화학식 하나로 표현 가능)
• 혼합물: 성분 물질 각각의 성질이 그대로 유지됨, 화학식 하나로 표현 불가능 (끓는점·녹는점이 일정하지 않음)
